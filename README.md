# WordHelper 📝

WordHelper is a lightweight Chrome extension that helps you improve your writing experience right in the browser. It provides quick access to definitions, synonyms, antonyms, and word suggestions without breaking your workflow.

---

## 🚀 Features

* 🔍 **Instant Definitions** – Highlight a word and get its meaning instantly.
* 🗣️ **Synonyms & Antonyms** – Expand your vocabulary with quick word alternatives.
* 📖 **Quick Dictionary** – Search any word directly from the extension popup.
* ✍️ **Writing Assistance** – Helpful suggestions for better word choice.
* ⚡ **Fast & Lightweight** – Minimal design with maximum usability.

---

## 📦 Installation

### From Source (Developer Mode)

1. Clone this repository or download the ZIP file.

   ```bash
   git clone https://github.com/your-username/wordhelper.git
   ```
2. Open **Chrome** and navigate to:

   ```
   chrome://extensions/
   ```
3. Enable **Developer Mode** (top-right toggle).
4. Click **Load unpacked**.
5. Select the `wordhelper` project folder.

The extension will now appear in your Chrome toolbar 🎉.

---

## 🔧 Usage

1. Click on the **WordHelper** icon in the toolbar.
2. Type or paste any word in the search box.
3. Instantly view **meanings, synonyms, and antonyms**.
4. Alternatively, **highlight any word** on a webpage, right-click, and choose **"WordHelper: Look up"** from the context menu.
